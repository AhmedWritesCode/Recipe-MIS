Web-based Healthy Recipe Management System

  • Overview
  • User Manual
  • Report's List of Contents

A secure web platform for users to manage healthy recipes. Users can create, view, edit, and delete recipes with ingredients, steps, and categories. Sorting, filtering, and responsive design ensure easy organization and access across devices. SQL database stores data securely with validation and error handling.

Scenario:

User registers, logs in, and creates a healthy breakfast smoothie recipe with ingredients and steps. They can view, edit, or delete recipes, sort by category, and filter by ingredients. An admin manages user accounts and monitors the recipe database.

Key Features:

  • User Authentication: Allow users to register and log in securely.
  • Task Management: Users can view, add, edit, and delete recipes with details.
  • Sorting and Filtering: Implement options for sorting and filtering recipes.
  • Responsive Design: Ensure the application works well on various devices.
  • Database Integration: Use MySQL to store user information and recipe data.
  • Validation and Error Handling: Ensure data integrity and display meaningful error messages.
  • User-Friendly Interface: Design an intuitive and user-friendly interface.

Technologies:

HTML, CSS, JavaScript, PHP, and SQL

Getting Started

  1. Open the web application in your browser.
  2. Register a new account or log in if you already have one.

Managing Recipes

  1. To create a new recipe, click on the "Add Recipe" button.
  2. Fill in the recipe details including title, category, ingredients, and steps.
  3. Click "Save" to add the recipe to your collection.

Viewing and Editing Recipes

  1. To view your recipes, navigate to the "My Recipes" section.
  2. Click on a recipe title to view details.
  3. To edit a recipe, click on the "Edit" button, make changes, and save.

Deleting Recipes

  1. In the "My Recipes" section, click the "Delete" button next to the recipe you want to remove.

Sorting and Filtering Recipes

  1. Use the sorting and filtering options in the "My Recipes" section to organize your recipes.

Admin Panel

  1. Log in as an admin to access the admin panel.
  2. Manage user accounts and monitor the recipe database.

Troubleshooting

If you encounter any issues, contact support at support@healthyrecipes.com.
